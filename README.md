# Pure-Bite-Box-Fresh-Fruit-Healthy-Food-Delivery-Website
FRUUT is a modern, responsive fruit &amp; healthy food delivery website built to provide a fast, simple, and clean shopping experience. Features include product catalog, cart UI, mobile-friendly pages, and a scalable structure ready for API and backend integration.
